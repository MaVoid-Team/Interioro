# Technical Stack

## Frontend
- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Internationalization:** i18n implementation (based on `frontend/i18n`)

## Backend
- **Runtime:** Node.js
- **Language:** TypeScript
- **Framework:** Express.js
- **ORM:** Sequelize
- **Testing:** Jest

## Infrastructure & Deployment
- **Containerization:** Docker
- **Web Server:** Nginx
- **Database:** Relational Database (via Sequelize)

## Design Implementation
- **Tokens:** Referenced in `resources/design-tokens.json`
- **Guidelines:** Defined in `design.md`
